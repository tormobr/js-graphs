Welcome to the help page!

Purpose:

The purpose of this website is to be able to play around with different pathfinding algorithms. Understanding the theory is of-course important, but can be tiresome. On this site all the tiresome math stuff happens under the hood, and you can enjoy the beautiful results!

What are those buttons?

The buttons are devided into three different sections.

  • Algorithm stuff
    • Here you can do things such as execute different pathfinding algorithms. Some examples are BFS, DFS and A*. In this section you can also clear the grid, generate maps, etc.
  • Painting stuff
    • You are equiped with a couple of painting brushes (for free). These can be used to draw cells, remove cells and change start and target positions
  • More algorithm stuff
    • This sections allows you to tweek more detailed parameters. You can for example change the heuristic function being used for A* search. Changing these parameters affects how the algorithms from the first section behaves
